1. Для запуска проекта ввести команду "docker-compose up" 
2. Для проверки результата в терминале ввести в терминале  "curl http://localhost" или в адресной строке браузера "http//:localhost (можно указать порт :80, можно не указывать)
Ожидамый резульат "Hello from Effective Mobile!"
3. Схема работы, контенер nginx принимает запрос, так как в новом конфиге указано перенаправлять запрос на http://backend контейнер с нашим приложением принимает и обратывает запрос и отдает результат. Перенаправление с http://backend таким адресом работает так как в dockercompose сервисы общаются по доменным именам, наш сервис с приложением называется как раз backend.



Дополнительно!

Так же мы можем запустить данный проект в режиме разззработки, для этого необходимо добавить watch в наш docker-compose.yaml
После чего нужно запустить проект командой "docker compose watch" и после любого изменения фаила app.py (например поменять "Hello" на "Привет") контенер пересоберется и вы увидите сразу же результат изменений. 
services:
  backend:
    build: ./backend
    container_name: effective-backend
    develop:
      watch:
        - path: ./backend/app.py
          action: sync
          target: /app/


          
  nginx:
    build: ./nginx
    container_name: effective-nginx
    ports:
      - "80:80"
